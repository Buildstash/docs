# Buildstash Docs

Documentation for [Buildstash](https://buildstash.com) - a platform for managing software binaries and releases.

Content is organized within 3 parent sections:
- Platform (core docs for Buildstash)
- API
- Integrations

These docs are built using the excellent [Fumadocs](https://fumadocs.dev/).

## Contributions

We welcome contributions to improve these docs! If you spot a mistake please open an issue or submit a pull request.

General improvements to the docs are also welcomed via PRs.

## Running locally

Run development server:

```bash
bun run dev
# or
npm run dev
# or
pnpm dev
# or
yarn dev
```

Open http://localhost:3000 with your browser to test.
