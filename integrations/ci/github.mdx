If you're using GitHub Actions sending your build artifacts into Buildstash is easy.

## Create your application API key

<Steps>
  <Step title="Create API key">
    In your JumpCloud admin portal, navigate to "SSO Applications", and select "Add New Application".

    ![](/SCR-20241222-qafi.png)

    Select "Custom Application".
  </Step>

  <Step title="Safely store for use">
    Then check "Manage Single Sign-On", and select "Configure SSO with OIDC".

    ![](/SCR-20241222-qayf.png)
  </Step>
</Steps>

## Add Buildstash action to your workflow

<Steps>
  <Step title="Install Buildstash action">
    Go to the <a href="https://github.com/marketplace/actions/buildstash" target="_blank">Buildstash action page on the GitHub Marketplace</a>. You can copy and paste the provided code into your workflow yaml script.

    ![](/SCR-20241222-qebl.png)
  </Step>

  <Step title="Tweak input values accordingly">
    Tweak input values accordingly.

    ![](/SCR-20241222-qelj.png)
  </Step>

  <Step title="Watch your builds appear in Buildstash!">
    Watch your builds appear in Buildstash!

    ![](/SCR-20241222-qerz.png)
  </Step>
</Steps>
